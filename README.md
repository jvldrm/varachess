# Varachess

Varachess provides a new experience in the game of chess, giving an incentive to the users of the dapp, that is, they get a reward for playing and are referred to their friends. As well as tools to improve strategy, anticipate movements (saving time) and many more possibilities to make the experience more fun.
The philosophy behind VaraChess is simple but powerful: "Play to win" and "Learn to win".

## Installation
To get started with varachess, follow these steps:

Clone the repository: git clone https://github.com/jvldrm/varachess
Navigate frontend or smart contracts directory: cd xxxxxxxx
Follow the instructions of each readme included in each directory.
